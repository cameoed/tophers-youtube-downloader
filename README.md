# Topher's YouTube Downloader

![Topher's YouTube Downloader](assets/youtube-downloader.jpg)

A simple YouTube downloader for Mac and Windows.

Download `dist/YouTube Downloader Release.zip`, unzip it, then open the Mac or Windows folder.

The app may ask you to install `yt-dlp` and `ffmpeg` the first time you use it.

## Mac First Open

macOS may say it cannot verify the app or install helper.

To open the app:

1. Open the `Mac` folder.
2. Control-click `YouTube Downloader`.
3. Choose `Open`.
4. Choose `Open` again if macOS asks.

To run the install helper:

1. Open the `Mac` folder.
2. Control-click `Install Required Tools.command`.
3. Choose `Open`.
4. Choose `Open` again if macOS asks.

If macOS only shows `Move to Trash` and `Done`, click `Done`, then go to `System Settings > Privacy & Security`, scroll to `Security`, and click `Open Anyway`. After that, try again with Control-click > Open.

Please only download content you have the rights or permission to save.
